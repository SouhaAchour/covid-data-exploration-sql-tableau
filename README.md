# COVID-19 Data Exploration & Tableau Dashboard

##  Project Overview
This project explores global COVID-19 data using SQL and visualizes key insights through an interactive Tableau dashboard.

The goal is to analyze infection rates, death counts, and vaccination trends to better understand the impact of COVID-19 across different countries and continents.

---

##  Objectives
- Explore COVID-19 dataset using SQL
- Clean and transform data
- Calculate key metrics (cases, deaths, infection rates)
- Use advanced SQL techniques (CTE, joins, window functions)
- Build an interactive Tableau dashboard
